\# Cloud-Native CI/CD Pipeline with Kubernetes Monitoring



\## Project Overview



This project demonstrates a complete DevOps workflow using GitHub, Jenkins, Docker, Kubernetes, Prometheus, and Grafana.



\## Architecture



GitHub → Jenkins → Docker → Docker Hub → Kubernetes → Prometheus → Grafana



\## Technologies Used



\* Git \& GitHub

\* Jenkins

\* Docker

\* Docker Hub

\* Kubernetes

\* Helm

\* Prometheus

\* Grafana

\* Python Flask

\* Linux



\## Features



\* CI/CD Pipeline using Jenkins

\* Dockerized Flask Application

\* Docker Hub Image Registry

\* Kubernetes Deployment with Replicas

\* Rolling Updates

\* Monitoring with Prometheus

\* Visualization with Grafana

Successfully deployed and monitored a cloud-native application using modern DevOps tools and practices.
